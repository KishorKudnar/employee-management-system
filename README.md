# Employee Management System

This is a Full Stack Web Application to manage employees.

## 🔧 Technologies Used

- 💻 Frontend: React.js
- 🔙 Backend: Spring Boot (Java)
- 🛢️ Database: PostgreSQL (or H2)
- 📄 PDF Generation
- ✉️ Email Notification (optional)
